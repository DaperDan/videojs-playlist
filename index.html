videojs-playlist

The playlist plugin provides multiple track playlist. The two main features are 1) manual clicking between tracks; 2) continuous play which loops back to the first track when reach the end of the playlist.

Required class and id names

 
    id="XXX-vjs-playlist" //playlist wrapper ID that is specific to the instantiated videojs object ID, e.g., id="audio-playlist-vjs-playlist"

class="vjs-track" // tracks className is required
